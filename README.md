# World Cup 2026 Bracket ⚽️

A stunning, real-time, and fully responsive radial tournament bracket for the FIFA World Cup 2026. Built with React, TypeScript, and Vite.

![Bracket Preview](./public/vite.svg) *// Replace with actual screenshot*

## Features 🌟
- **Radial Orthogonal Design**: A perfectly symmetrical circular tree featuring elegant T-junctions and SVG arcs.
- **Real-Time Data**: Fetches live match states, scores, and updates every 60 seconds from TheSportsDB.
- **Timezone Detection**: Automatically detects and displays match times in your local timezone, with a manual override selector.
- **Live Match Highlights**: Glowing red indicators and pill badges for matches currently in progress.
- **Premium UI**: Glassmorphism, beautiful curved SVG text paths, custom Google Fonts (Outfit & Inter), and a sleek dark mode.

## Tech Stack 🛠️
- **React 18** (Hooks, Context, Memoization)
- **TypeScript** (Strict typing for API and SVG paths)
- **Vite** (Lightning-fast HMR and build)
- **Vanilla CSS** (Custom CSS variables, keyframe animations, no heavy frameworks)

## Installation 🚀

1. Clone the repository:
   ```bash
   git clone https://github.com/Ezarwebmaster/WC26.git
   cd WC26
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
